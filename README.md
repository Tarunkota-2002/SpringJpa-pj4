# 🚀 Spring Data JPA Demo Project


**📌** **Overview**
=====================

This project demonstrates how to perform CRUD and custom database operations using Spring Data JPA in a layered architecture.

**It includes:**

Repository-based queries

Service layer abstraction

Custom query methods

Filtering and deletion operations


**🧩** **Tech Stack**
=======================

☕ Java 17+

🌱 Spring Boot

🗄️ Spring Data JPA (Hibernate)

🐬 ORACLESQL / Any RDBMS


**🧠 Key Concepts Used**
==========================

✔️ Spring Data JPA Method Naming Convention

✔️ Dependency Injection (@Autowired)

✔️ Layered Architecture (Controller → Service → Repository)

✔️ Custom Queries using @Query

✔️ Bean Management using ApplicationContext



🧪 Maven
